## Getting Started

1. Clone this project (https://github.com/BrandonJhonn/flutter_app_diptsv3_todo.git)
2. After you must run: 

```
$ flutter pub get
$ flutter run
```

3. To run the test scenaries, you must run:

```
$ flutter driver --target=test_driver/bdd.dart
```
